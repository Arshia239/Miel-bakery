# 🥐 Miel Bakery

A responsive bakery website built as **Project 2** of the DecodeLabs Frontend Development Internship (Batch 2026).

🌐 **Live Demo:** [miel-bakery.vercel.app](https://miel-bakery.vercel.app/)

---

## 📌 About the Project

Miel Bakery is a fully responsive, mobile-first webpage for a fictional artisan bakery. The goal was to master **CSS responsive design techniques** — ensuring the layout looks great on every screen size, from mobile phones to large desktop monitors.

---

## ✨ Features

- ✅ Responsive navigation with **hamburger menu** (Popover API — no JS required)
- ✅ **CSS Grid** for page layout (macro structure)
- ✅ **Flexbox** for component-level alignment
- ✅ **CSS Media Queries** with mobile-first approach
- ✅ **Fluid typography** using `clamp()`
- ✅ **Fluid spacing** using `%`, `rem`, and `vw` units
- ✅ Scrolling marquee strip (pure CSS animation)
- ✅ Real food images from Unsplash
- ✅ Hover effects and smooth transitions
- ✅ Accessible touch targets (min 44×44px)

---

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| `0px – 539px` | Mobile — single column |
| `540px – 767px` | Tablet small — 2 columns |
| `768px – 1023px` | Tablet — 2 column hero |
| `1024px+` | Desktop — 4 column grid |
